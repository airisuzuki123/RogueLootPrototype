# 阶段 5.5 职业连续选择审计报告

本报告模拟每个职业开局后最可能出现的前三个首选方向，并继续计算第二次选择后的候选集中度。它用于发现前两关出货是否快速塌缩，不代表玩家必然拿到对应技能。

## 审计口径

- 使用与 GameManager 相同的稀有度、重复、协同、标签、冲突和职业权重公式。
- 对开局权重最高的 3 个首选分别建立分支，再选择该分支下权重最高的第二张技能。
- 只把 `bulk`, `agile`, `pierce`, `blast`, `chain`, `close` 计为正式构筑路线；辅助标签不计入路线覆盖。
- 前 8 名中至少保留 2 个清群入口、覆盖 3 条正式路线；第一名权重占前 8 名总权重超过 30% 时记为集中度警告。

## 结论摘要

- 巨躯炮台：通过；开局 清群 4 / 路线 4 / 首位占比 17%；开局清群概率至少 70%；风险分支 0/3；最高集中度 23%。
- 沉稳射手：通过；开局 清群 2 / 路线 4 / 首位占比 19%；开局清群概率至少 64%；风险分支 0/3；最高集中度 26%。
- 贴身刃卫：需关注；开局 清群 3 / 路线 3 / 首位占比 27%；开局清群概率至少 37%；风险分支 2/3；最高集中度 33%。
- 游走电弧：通过；开局 清群 2 / 路线 3 / 首位占比 19%；开局清群概率至少 75%；风险分支 0/3；最高集中度 23%。
- 重弹爆破：需关注；开局 清群 3 / 路线 5 / 首位占比 25%；开局清群概率至少 72%；风险分支 1/3；最高集中度 34%。

## 候选调平方案预测

该预测只覆盖权重与标签调整，不修改项目实际职业数据。

- 贴身刃卫：风险分支 0/3，最高集中度 17%，开局清群概率至少 39%；开局候选 近身刀环(74)、脉冲场(74)、巨体回响(72)、清弹屏障(71)、折光护盾(71)。
- 重弹爆破：风险分支 0/3，最高集中度 29%，开局清群概率至少 69%；开局候选 裂片爆破(116)、爆裂核心(90)、重压弹芯(88)、穿透弹芯(62)、锚定释放(50)。
