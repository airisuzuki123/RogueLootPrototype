# 阶段 5.5 职业连续选择审计报告

本报告模拟每个职业开局后最可能出现的前三个首选方向，并继续计算第二次选择后的候选集中度。它用于发现前两关出货是否快速塌缩，不代表玩家必然拿到对应技能。

## 审计口径

- 使用与 GameManager 相同的稀有度、重复、协同、标签、冲突和职业权重公式。
- 对开局权重最高的 3 个首选分别建立分支，再选择该分支下权重最高的第二张技能。
- 只把 `bulk`, `agile`, `pierce`, `blast`, `chain`, `close` 计为正式构筑路线；辅助标签不计入路线覆盖。
- 前 8 名中至少保留 2 个清群入口、覆盖 3 条正式路线；第一名权重占前 8 名总权重超过 30% 时记为集中度警告。
- 开局三选一的前两个路线槽至少出现一张清群技能的估算概率不得低于 55%。

## 结论摘要

- 巨躯炮台：通过；开局 清群 4 / 路线 4 / 首位占比 17%；开局清群概率至少 91%；风险分支 0/3；最高集中度 23%。
- 沉稳射手：通过；开局 清群 2 / 路线 4 / 首位占比 21%；开局清群概率至少 88%；风险分支 0/3；最高集中度 27%。
- 贴身刃卫：通过；开局 清群 2 / 路线 3 / 首位占比 14%；开局清群概率至少 58%；风险分支 0/3；最高集中度 17%。
- 游走电弧：通过；开局 清群 2 / 路线 3 / 首位占比 19%；开局清群概率至少 92%；风险分支 0/3；最高集中度 23%。
- 重弹爆破：通过；开局 清群 3 / 路线 5 / 首位占比 21%；开局清群概率至少 91%；风险分支 0/3；最高集中度 29%。
