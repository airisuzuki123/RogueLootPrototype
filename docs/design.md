# 设计草案

## 核心循环

1. 玩家进入一局。
2. 敌人持续生成并追踪玩家。
3. 玩家击杀敌人，获得金币、经验或装备。
4. 玩家通过升级和装备形成构筑。
5. 一局结束后保留部分局外资源，用于长期成长。

## 最小可玩版本

- 一个玩家角色。
- 多种基础敌人：普通、小型快速、坦克、远程。
- 一种自动攻击方式。
- 金币掉落。
- 装备掉落和背包保存。
- 背包内同槽位装备比较、装备选择和分解金币。
- 武器形态：聚能、散射、穿透、爆裂。
- 基础装备词缀：伤害、生命、攻速、移速、暴击率、额外投射物、穿透、爆裂范围。
- 基础装备槽位：武器、护甲、饰品。
- 经验与升级三选一。
- HUD 状态显示。
- 一局计时或死亡结算。
- 基础战斗反馈：伤害数字、命中特效、受击闪烁、击退、敌人血条。

## 当前敌人生态

- 普通敌人：标准速度、生命和接触伤害。
- 小型快速敌人：低生命、高速度，用于制造追击压力。
- 坦克敌人：高生命、低速度、高接触伤害，用于制造空间压迫。
- 远程敌人：保持距离并发射投射物，用于打破单纯绕圈。

## 后续系统

- 装备稀有度：普通、魔法、稀有、传奇。
- 装备词缀：生命、伤害、攻速、移速、暴击、特殊投射物、吸血。
- 背包与装备选择：拾取后可比较、保留或分解。
- 升级奖励：三选一属性或技能强化。
- 关卡事件：精英怪、宝箱、商店、Boss。

## 当前背包与装备流程

- 装备掉落后，玩家碰到掉落物会直接放入背包，不再立即暂停战斗。
- 玩家按 `B` 打开背包时暂停战斗，再按 `B` 或 `Esc` 关闭背包。
- 背包内装备按照品质和等级排序，同品质同等级时再按评分排序。
- 背包列表显示品质颜色、槽位、形态或类型、等级、评分和同槽位评分差。
- 背包详情区优先显示推荐结论，并并排显示当前同槽位装备和选中装备。
- 背包面板显示当前同槽位装备、选中装备、词缀、评分、评分变化和核心属性变化。
- 玩家可以从背包装备武器、护甲或饰品，也可以分解背包内装备获得金币。
- 分解金币由品质基础值和装备评分共同决定，避免低评分稀有装备与高评分普通装备价值完全脱节。
- 当前背包不设容量上限，先用于验证拾取后延迟决策的节奏。

## 当前装备槽位

- 武器：决定法杖形态，并可出现伤害、攻速、暴击、额外投射物、穿透和爆裂范围词缀。
- 护甲：偏生存和机动，当前可出现最大生命、移动速度和攻击速度词缀。
- 饰品：偏构筑补强，当前可出现暴击、攻速、移速、额外投射物、穿透和爆裂范围词缀。
- 同一时间每个槽位只能装备一件装备，替换时旧装备会回到背包。

## 当前武器形态

- 聚能法杖：单发伤害更高，适合高伤害和暴击构筑。
- 散射法杖：额外发射 2 枚投射物，单枚伤害较低，适合覆盖近距离敌群。
- 穿透法杖：投射物可额外穿透 1 名敌人，适合处理排队追击的敌人。
- 爆裂法杖：命中时造成小范围爆裂伤害，适合清理聚集敌人。

## 当前装备词缀

- 伤害：提高玩家投射物基础伤害。
- 最大生命：提高最大生命并补足同等生命值，替换装备时会回收该生命上限。
- 攻击速度：按百分比缩短自动攻击间隔。
- 移动速度：提高玩家移动速度。
- 暴击率：投射物发射时按概率造成双倍伤害，并显示更醒目的命中反馈。
- 额外投射物：每次自动攻击额外发射投射物，会和散射法杖及升级奖励叠加。
- 穿透：提高投射物额外穿透敌人的次数，会和穿透法杖叠加。
- 爆裂范围：让投射物命中时造成小范围伤害；如果武器本身是爆裂法杖，则进一步扩大爆裂并略微提高范围伤害比例。

## 当前掉落与经济调校

- 装备掉落概率会随玩家等级小幅提升，但当前上限仍保持克制，避免每个掉落都变成装备选择。
- 金币掉落数量会随玩家等级缓慢提高，用于补偿后期分解之外的经济来源。
- 分解金币使用装备品质基础值加评分换算值，并设置上限，避免单件装备经济收益过大。
