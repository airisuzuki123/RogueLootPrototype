# 设计草案

## 核心循环

1. 玩家进入一局。
2. 敌人持续生成并追踪玩家。
3. 玩家击杀敌人，直接获得经验，并可能额外获得金币或掉落装备。
4. 玩家通过升级和装备形成构筑。
5. 一局结束后保留部分局外资源，用于长期成长。

## 最小可玩版本

- 一个玩家角色。
- 多种基础敌人：普通、小型快速、坦克、远程。
- 一种自动攻击方式。
- 金币获取。
- 击杀获得经验并触发升级。
- 装备掉落和背包保存。
- 背包内同槽位装备比较、装备选择和分解金币。
- 武器形态：聚能、散射、穿透、爆裂。
- 基础装备词缀：伤害、生命、攻速、移速、暴击率、金币获取、额外投射物、穿透、爆裂范围。
- 基础装备槽位：武器、头盔、护甲、鞋子、项链、戒指。
- 经验与升级三选一。
- HUD 状态显示。
- 一局计时或死亡结算。
- 基础战斗反馈：伤害数字、命中特效、受击闪烁、击退、敌人血条。

## 当前敌人生态

- 普通敌人：标准速度、生命和接触伤害。
- 小型快速敌人：低生命、高速度，用于制造追击压力。
- 坦克敌人：高生命、低速度、高接触伤害，用于制造空间压迫。
- 远程敌人：保持距离并发射投射物，用于打破单纯绕圈。

## 后续系统

- 装备稀有度：普通、魔法、稀有、传奇。
- 装备词缀：生命、伤害、攻速、移速、暴击、金币获取、特殊投射物。
- 背包与装备选择：拾取后可比较、保留或分解。
- 升级奖励：三选一属性或技能强化。
- 关卡事件：精英怪、宝箱、商店、Boss。

## 当前背包与装备流程

- 敌人死亡后直接给予经验，并可能触发升级选择，减少弹幕战斗中的拾取物干扰。
- 金币直接加入账户；装备使用品质颜色大菱形并标槽位字。
- 装备掉落后，玩家碰到掉落物会直接放入背包，不再立即暂停战斗。
- 玩家按 `B` 打开背包时暂停战斗，再按 `B` 或 `Esc` 关闭背包。
- 背包内装备按照品质和等级排序，同品质同等级时再按评分排序。
- 背包容量为 36 格，容量满后装备掉落无法继续拾取，掉落物保留在场上并显示背包已满提示。
- 玩家清出背包容量后，如果仍站在未拾取的装备掉落物上，关闭背包恢复战斗后会自动再次尝试拾取。
- 背包主界面左侧显示当前穿戴槽位和构筑总览，右侧显示容量、品质分解勾选、分解选中按钮和背包装备图标网格，初始不显示分页签或筛选栏。
- 当前穿戴槽位固定为武器、头盔、护甲、鞋子、项链、戒指 6 件装备。
- 背包装备图标显示品质颜色边框、槽位图形、等级和评分，选中时使用高亮边框，列表仍按照品质、等级、评分排序。
- 构筑总览汇总当前穿戴装备的总评分、武器形态和装备词缀总加成，用于快速确认当前构筑方向。
- 点击当前穿戴或背包装备图标后，使用弹窗显示装备详情；背包装备详情中当前装备和选中装备左右并列显示，详情内容区域可滚动，底部装备和分解按钮固定显示。
- 装备详情弹窗不使用右上角关闭按钮，点击弹窗外区域或按 `Esc` 关闭。
- 点击背包装备时，弹窗优先显示左右并列的当前同槽位装备和选中装备，并在下方显示推荐结论、评分变化和核心属性变化。
- 玩家可以从背包装备任意槽位装备，也可以卸下当前穿戴装备放回背包。
- 玩家可以单件分解背包装备，也可以勾选一个或多个指定品质后分解选中品质获得金币。
- 分解金币由品质基础值和装备评分共同决定，避免低评分稀有装备与高评分普通装备价值完全脱节。

## 当前装备槽位

- 武器：决定法杖形态，并可出现伤害、攻速、暴击、额外投射物、穿透和爆裂范围词缀。
- 头盔：偏生存和暴击，当前可出现最大生命和暴击率词缀。
- 护甲：偏生存和经济，当前可出现最大生命、移动速度和金币获取词缀。
- 鞋子：偏机动和节奏，当前可出现移动速度、攻击速度和金币获取词缀。
- 项链：偏构筑补强，当前可出现暴击率、爆裂范围和金币获取词缀。
- 戒指：偏输出节奏，当前可出现攻击速度、额外投射物、穿透和金币获取词缀。
- 同一时间每个槽位只能装备一件装备，替换时旧装备会回到背包。

## 当前武器形态

- 聚能法杖：单发伤害更高，适合高伤害和暴击构筑。
- 散射法杖：额外发射 2 枚投射物，单枚伤害较低，适合覆盖近距离敌群。
- 穿透法杖：投射物可额外穿透 1 名敌人，适合处理排队追击的敌人。
- 爆裂法杖：命中时造成小范围爆裂伤害，适合清理聚集敌人。

## 当前升级与武器形态联动

- 玩家升级时，如果当前已经装备武器，升级池会额外加入一个与当前武器形态匹配的专精选项。
- 聚能专精：投射物伤害 +8，强化单发和暴击收益。
- 散射专精：每次攻击投射物 +1，强化覆盖和近距离清场。
- 穿透专精：投射物穿透 +1，强化队列敌人处理能力。
- 爆裂专精：爆裂范围 +28，强化聚集敌人的范围清理。
- 专精在选择时立即固定生效，之后更换武器不会移除已经获得的升级效果。

## 当前装备词缀

- 伤害：提高玩家投射物基础伤害。
- 最大生命：提高最大生命并补足同等生命值，替换装备时会回收该生命上限。
- 攻击速度：按百分比缩短自动攻击间隔。
- 移动速度：提高玩家移动速度。
- 暴击率：投射物发射时按概率造成双倍伤害，并显示更醒目的命中反馈。
- 吸血：旧版装备兼容逻辑保留，但新掉落装备不再投放该词缀。
- 金币获取：提高玩家获得金币的数量，当前会影响击杀金币收益和装备分解收益。
- 额外投射物：每次自动攻击额外发射投射物，会和散射法杖及升级奖励叠加。
- 穿透：提高投射物额外穿透敌人的次数，会和穿透法杖叠加。
- 爆裂范围：让投射物命中时造成小范围伤害；如果武器本身是爆裂法杖，则进一步扩大爆裂并略微提高范围伤害比例。

## 当前掉落与经济调校

- 敌人死亡直接给予经验，用于承担主要成长节奏。
- 装备掉落概率会随玩家等级小幅提升，但当前上限保持克制，避免满屏装备或每个掉落都变成装备选择。
- 击杀金币收益会随玩家等级缓慢提高，用于补偿后期分解之外的经济来源。
- 分解金币使用装备品质基础值加评分换算值，并设置上限，避免单件装备经济收益过大。

## 当前单局阶段节奏

- 单局计时会在正常战斗中推进，打开背包或升级选择时暂停。
- 第一版阶段结构为初始清场、追击加压、混编压迫、密集来袭、终局试炼。
- HUD 显示单局时间、当前阶段、阶段剩余时间、下一阶段预告和当前阶段目标。
- 阶段会调整刷怪间隔、单次刷怪数量、敌人解锁等级和敌人权重。
- 每个阶段有独立击杀目标，HUD 显示当前阶段进度和奖励预览。
- 达成阶段击杀目标后会立即发放少量金币、经验或生命回复，并弹出目标完成提示。
- 进入新阶段时会弹出阶段转换提示，帮助玩家感知压力曲线变化。
- 阶段剩余 10 秒时会弹出倒计时提示，提前告知下一阶段或本局完成。
- 终局试炼有固定持续时间，撑过后会完成本局并显示完成结算；死亡仍会提前结束本局。
- 当前阶段 3 只建立局内节奏骨架，不加入局外成长、永久强化或长期资源消耗。

## 阶段 3 调整方向：弹幕式怪物与走位躲避

- 后续关卡体验转向弹幕式躲避原型，压力主要来自怪物发射的可读弹幕，而不是普通敌人贴身数量。
- 目标体验参考弹幕射击和几何竞技场：高亮几何弹体、环形扩散、螺旋、扇形扫射、弹幕墙和封闭场地穿缝走位。
- 装备构筑继续提供输出、生存和移动收益，但阶段 3 的优先目标是让玩家通过走位、预判弹道和寻找安全缝隙来生存。
- 现有时间阶段、阶段目标、奖励、倒计时和完成结算作为承载弹幕节奏的流程骨架保留。
- 远程敌人已接入第一版阶段弹幕池：直线、集束、扇形、环形、交叉、螺旋、双层环、花形和弹幕墙。
- 阶段会从对应弹幕池中抽取发射模式，避免每个阶段只有一种固定弹幕。
- 弹幕型敌人已加入游走扫射者和阵列发射者，分别偏向扫射穿缝与阵地扩散压力。
- 后续阶段 3 开发继续调校弹幕型敌人比例、阶段主题和画面可读性。

## 阶段 2 收尾验收清单

- 装备拾取进入背包，不弹出即时选择面板，不频繁打断战斗。
- 击杀敌人直接获得经验和金币，装备掉落外观清晰可读。
- 打开背包时玩家、敌人、玩家投射物、敌人投射物和刷怪流程暂停。
- 背包关闭后战斗继续，`Esc` 优先关闭装备详情弹窗。
- 背包排序保持品质 > 等级 > 评分。
- 背包显示 36 格容量，满容量后装备无法继续拾取；清出容量后可重新拾取保留在场上的装备。
- 当前穿戴区固定展示武器、头盔、护甲、鞋子、项链、戒指。
- 背包主界面保持左侧穿戴区和构筑总览、右侧装备图标网格，不默认显示分页签或筛选栏。
- 点击穿戴装备或背包装备图标后弹出详情，不在主界面常驻大段详情文本；背包装备对比左右并列，详情内容过长时可滚动。
- 点击装备详情弹窗外区域可以关闭弹窗。
- 背包装备详情显示推荐结论、当前同槽位装备、选中装备、评分变化和核心属性变化。
- 从背包装备任意槽位装备时，旧装备回到背包并重新参与排序。
- 当前穿戴装备可以卸下回到背包，背包满时无法卸下。
- 单件分解或勾选指定品质后分解选中品质时获得金币，金币收益受到品质、评分和金币获取词缀影响。
- 六槽位词缀加成能正确影响玩家属性，替换装备时旧加成被移除。
- 武器形态和形态专精在战斗中有可感知差异。
- 构筑总览能清楚展示当前穿戴装备带来的核心加成。
- 当前阶段只要求原型可验收，不要求最终内容量和最终平衡。
