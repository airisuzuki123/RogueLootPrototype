# 设计草案

## 核心循环

1. 玩家进入一局。
2. 敌人持续生成并追踪玩家。
3. 玩家击杀敌人，获得金币、经验或装备。
4. 玩家通过升级和装备形成构筑。
5. 一局结束后保留部分局外资源，用于长期成长。

## 最小可玩版本

- 一个玩家角色。
- 多种基础敌人：普通、小型快速、坦克、远程。
- 一种自动攻击方式。
- 金币掉落。
- 武器掉落和自动装备。
- 武器拾取比较、装备选择和分解金币。
- 武器形态：聚能、散射、穿透、爆裂。
- 基础装备词缀：伤害、生命、攻速、移速、暴击率。
- 经验与升级三选一。
- HUD 状态显示。
- 一局计时或死亡结算。
- 基础战斗反馈：伤害数字、命中特效、受击闪烁、击退、敌人血条。

## 当前敌人生态

- 普通敌人：标准速度、生命和接触伤害。
- 小型快速敌人：低生命、高速度，用于制造追击压力。
- 坦克敌人：高生命、低速度、高接触伤害，用于制造空间压迫。
- 远程敌人：保持距离并发射投射物，用于打破单纯绕圈。

## 后续系统

- 装备稀有度：普通、魔法、稀有、传奇。
- 装备词缀：生命、伤害、攻速、移速、暴击、特殊投射物、吸血。
- 背包与装备选择：拾取后可比较、保留或分解。
- 升级奖励：三选一属性或技能强化。
- 关卡事件：精英怪、宝箱、商店、Boss。

## 当前装备拾取流程

- 武器掉落后，玩家碰到掉落物会暂停战斗并打开比较面板。
- 面板显示当前武器、新武器、词缀、评分和评分变化。
- 玩家可以装备新武器，也可以保留当前武器并分解新武器获得金币。
- 当前仍不保留背包库存，先用即时选择验证装备决策节奏。

## 当前武器形态

- 聚能法杖：单发伤害更高，适合高伤害和暴击构筑。
- 散射法杖：额外发射 2 枚投射物，单枚伤害较低，适合覆盖近距离敌群。
- 穿透法杖：投射物可额外穿透 1 名敌人，适合处理排队追击的敌人。
- 爆裂法杖：命中时造成小范围爆裂伤害，适合清理聚集敌人。

## 当前装备词缀

- 伤害：提高玩家投射物基础伤害。
- 最大生命：提高最大生命并补足同等生命值，替换装备时会回收该生命上限。
- 攻击速度：按百分比缩短自动攻击间隔。
- 移动速度：提高玩家移动速度。
- 暴击率：投射物发射时按概率造成双倍伤害，并显示更醒目的命中反馈。
