# 设计草案

## 核心循环

1. 玩家进入一局。
2. 敌人持续生成并追踪玩家。
3. 玩家击杀敌人，获得金币、经验或装备。
4. 玩家通过升级和装备形成构筑。
5. 一局结束后保留部分局外资源，用于长期成长。

## 最小可玩版本

- 一个玩家角色。
- 一种普通敌人。
- 一种自动攻击方式。
- 金币掉落。
- 武器掉落和自动装备。
- 基础装备词缀：伤害、生命、攻速。
- 经验与升级三选一。
- HUD 状态显示。
- 一局计时或死亡结算。

## 后续系统

- 装备稀有度：普通、魔法、稀有、传奇。
- 装备词缀：生命、伤害、攻速、移速、暴击。
- 背包与装备选择：拾取后可比较、保留或分解。
- 升级奖励：三选一属性或技能强化。
- 关卡事件：精英怪、宝箱、商店、Boss。
