# 阶段 5.5 数值调平实施记录

## 实施结论

五个职业已经完成开局出货、连续选择、代表性三技能运行时和前 3 关经济节奏校验。当前所有职业均满足：

- 开局候选前 8 至少包含 2 个真实清群技能。
- 前列候选至少覆盖 3 条正式构筑路线。
- 连续选择前三个分支风险为 0/3。
- 开局三选一的前两个路线槽至少出现一张清群技能的估算概率不低于 55%。
- 第一商店可以用确定性金币购买至少一张职业偏向技能。

## 通用调整

- 选择职业后立即获得一次职业偏向的开局三选一。
- 仅在这次开局三选一中，真实清群技能权重 x3；后续升级和商店恢复正常权重，不设置固定技能保底。
- 第一关目标金币提高到 25，技能阶段加价从第二间商店开始计算。
- 前 3 关最低形成 6 次有效构筑选择：1 次开局三选一、至少 3 次升级、2 次确定性技能购买。
- 分裂射击移速代价从 -25% 调整为 -20%。

## 职业调整

### 巨躯炮台

- 分裂射击 1.80 → 2.00。
- 穿透弹芯 1.55 → 1.80。
- 爆裂核心 1.35 → 1.50。
- 体积共鸣 1.55 → 1.35。
- 迟缓共鸣 1.45 → 1.30。
- 目标：先获得多投射、穿透或爆裂载体，再获得体积与迟缓增伤引擎。

### 沉稳射手

- 穿透弹芯 1.45 → 1.65。
- 贯穿增幅 1.35 → 1.50。
- 引导光束 1.25 → 1.35。
- 静立聚焦 1.80 → 1.70。
- 锚定释放 0.85 → 0.75。
- 目标：穿透清群先于静立支撑，但保留静立暴击作为职业核心。

### 贴身刃卫

- 不再把 `close_skill` 当作开局已拥有标签，只保留真实存在的 `shielded`。
- 近身刀环和脉冲场提高到 2.00。
- 护身锋刃降至 1.10，巨体回响降至 0.60，破阵反应降至 0.75。
- 护身锋刃不再计为直接清群技能。
- 目标：先获得刀环或脉冲场，再获取近身增伤和触发器支撑。

### 游走电弧

- 职业权重和核心倍率保持不变。
- 该职业的敏捷、连锁、穿透覆盖和开局清群概率均处于健康区间，避免无依据强化。

### 重弹爆破

- 裂片爆破权重从 1.80 降至 1.40。
- 爆裂范围上限从 72 提高到 112。
- 爆裂核心、裂片爆破和爆裂形态升级预览改为显示考虑当前值与上限后的实际增量。
- 暴击率构筑摘要按实际 95% 命中上限显示。
- 目标：保留裂片爆破第一候选地位，避免连续重复压过爆裂核心和重压弹芯。

## 最终审计

| 职业 | 开局清群概率下限 | 风险分支 | 最高集中度 | 开局前列方向 |
|---|---:|---:|---:|---|
| 巨躯炮台 | 91% | 0/3 | 23% | 多投射、穿透、体积 |
| 沉稳射手 | 88% | 0/3 | 27% | 穿透、静立、贯穿 |
| 贴身刃卫 | 58% | 0/3 | 17% | 刀环、脉冲、护盾 |
| 游走电弧 | 92% | 0/3 | 23% | 敏捷、回旋、连锁 |
| 重弹爆破 | 91% | 0/3 | 29% | 裂片、爆裂、重弹 |

贴身刃卫仍是开局清群概率最低的职业，但主动清群技能已经排在前两位，且不再被支撑牌挤压。继续提高会明显压缩近身护盾路线的选择空间，暂时保留为实战观察项。

## 验证范围

- 职业效果审计。
- 职业初始出货审计。
- 职业连续选择与开局清群概率审计。
- 阶段 5 经济节奏审计。
- 五职业代表性三技能运行时测试。
- 职业选择面板到开局三选一面板的真实主场景流程。
- Godot headless 项目启动验证。

本轮不调整刷怪间隔、敌人等级、弹速和弹幕池。先验证五个职业在前 3 关稳定成型后的实际关卡压力，再决定是否修改敌人侧曲线。
